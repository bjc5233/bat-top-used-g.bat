# bat-top-used-g.bat
g.bat执行跳转到某一界面。大类分为磁盘、系统工具、系统管理、控制面板、文件夹、网址


## 说明
当输入的命令在代码中找不到匹配的执行命令时，会调用tool_guessKey（需要Java环境）进行猜测，选择最接近的命令，提供参考
