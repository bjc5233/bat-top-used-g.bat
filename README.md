# batch-go


## 说明
g.bat快捷跳转到某一界面，节约手动操作的时间。大类分为磁盘、系统工具、系统管理、控制面板、文件夹、网址
* 当输入的命令无法匹配时，会调用tool_guessKey[需要Java环境]进行猜测，选择最接近的命令，提供参考
